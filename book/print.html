C/C++ 基礎

C語言在1972年被 丹尼斯·里奇 開發出來，在後來影響各個程式語言非常深遠。

而C++則是在1983年由 Bjarne Stroustrup 拓展C語言而開發出來的。

第一個C語言程式

第一個c語言程式

#include<stdio.h>

int main(){
    printf("Hello World!");
    return 0;
}

上面的程式碼可以在主控台(console)中輸出「Hello World!」，讓我們從第一行開始依序看起：

第一行：

#include<stdio.h>

在C/C++中，以「#」開頭的程式碼為「前置處理器」。

而「include」則是前置處理器的一種，用來引入標頭檔。

因為「printf」這個函式定義在「stdio.h」這個標頭檔中，所以必須引入才可以使用「printf」這個函數。
